  • record my english learning

20240114

  • 状语，修饰动词or形容词

    • 关系连词+分句，主谓宾 or 主谓宾，关系连词+分句
    • 当分句主语和主句主语一致的时候，可以省略分句的主语；如果分句的主语有be，be动词也可以省略；或者分句不省，主句用it
      • When happy，i can play computer

  • dreadful adj. 糟糕的

  • handicap n. 残疾

  • 非谓语

    • 无逗号（注意修饰的是谁）
      • i like to use computers having some software
      • i like to use computers bought by my dad
    • 有逗号（有可能修饰的是前面一句话）
      • I like to doing exercises, making me happy
      • I like to read books, writing some notes
      • I like to use computers, bouhgt by my dad

  • parallel n. / adj. 平行线，平行的

  • unparalleled adj. 空前的；无比的

  • 定语 adj. 修饰名词

    • 限定性定语从句

      • I have an apple that/which is red
      • I have a computer whose software is good
      • (当先行词从当宾语的时候，关系代词可以省略)
      • I like this book (that) you read

    • 非限定，只能用which,

      • 可能是修饰前面一句话，但不会是修饰前面的主语
      • I like to read books, which is useful for me

    • 带介词的定语从句

      • 注意前面 后面 都是完整的句子

      • I like this table on which there is a book

      • I like this book through which I can learn many things

  • As

    • 关系连词 - 引导状语从句
      • 因为，当
      • 正如， as someone pointed out, …
    • 介词 ( + 名词)
      • 作为， As a person, …

  • 系动词

    • prove/become/taste/sound/look/smell/feel + adj

  • by the mid-1980s, …

    • 到…时间点后

  • at a loss over 困惑，不知所措

    • A at a loss over B : A对B 不知所措

  • find（make） A (n. ) B（n. / v. / 介词 / adj.） 发现

  • as though 仿佛，好像

  • take sth for granted 把…当作理所应当

  • yielded to 屈服于
